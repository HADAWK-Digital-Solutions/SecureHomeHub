Users Page

Introduction

The Users feature of the Secure Home Hub allows you to manage user accounts, privileges, and access levels. This section provides details on user roles, superuser accounts, and how to configure user privileges effectively.

User Roles

  • Admin: Users with administrative privileges, allowing them to configure system settings, manage devices, and view detailed reports.
  • Standard User: Users with standard access, capable of controlling their assigned devices and receiving notifications.

Managing User Privileges

  • User Privileges: Access and modify the privileges assigned to each user. This includes defining which devices and features each user can control or access.
  • Superusers: View and manage superuser accounts. Superusers have elevated privileges and can perform advanced configurations and security-related tasks.
  • Super User Privileges: Define the specific privileges granted to superusers. This includes access to sensitive information and critical system settings.

Adding and Removing Users

To add or remove users, navigate to the Users page from the main menu. Here, you can add new users, assign roles, and configure their privileges. Additionally, you can remove users who no longer require access to the Secure Home Hub.

Help and Support

If you encounter issues or have questions related to user management, refer to the help documentation provided with the Secure Home Hub. Additionally, you can contact our support team for assistance.
