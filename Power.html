Power Options

Introduction

The Power Options feature of the Secure Home Hub allows you to control the power-related functions of the device. This section covers information on shutting down, locking, and restarting the Secure Home Hub.

Power Off

  • Power Off: Use this option to gracefully shut down the Secure Home Hub. This ensures that all processes are safely terminated before the device powers off.

Lock

  • Lock: Lock the Secure Home Hub to secure the current session. This prevents unauthorized access to the device while keeping background processes running.

Restart

  • Restart: Use this option to reboot the Secure Home Hub. Restarting the device can help apply system updates and resolve certain issues without a full shutdown.

Help and Support

If you encounter issues or have questions related to power options, refer to the help documentation provided with the Secure Home Hub. Additionally, you can contact our support team for assistance.
