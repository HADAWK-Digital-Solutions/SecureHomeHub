Network Page

Introduction

The Network feature of the Secure Home Hub provides tools to monitor and manage your home network. This section covers information about connected devices, internet connection status, connectivity testing, and intrusion detection with Snort.

Connected Devices

  • Connected Devices: View a list of all devices currently connected to your home network. This includes details such as device name, IP address, and connection status.

Internet Connection

  • Internet Connection: Check the status of your internet connection. This button provides information on whether the Secure Home Hub is currently connected to the internet.

Connectivity Test

  • Connectivity Test: Perform a connectivity test to ensure seamless communication between your Secure Home Hub and connected devices. This helps identify and troubleshoot any network issues.

Snort Intrusion Detection

  • Snort: Utilize Snort, an intrusion detection and prevention system, to monitor network traffic for potential security threats. This feature enhances the overall security of your home network.

Help and Support

If you encounter issues or have questions related to network management, refer to the help documentation provided with the Secure Home Hub. Additionally, you can contact our support team for assistance.
